# graphic

- 인쇄용 **CMYK**

  [cyan] [magenta] [yellow] [black] 

- 웹용 **RGB**

  [red] [green] [blue]



### 해상도 

- dpi(dot per inch)   : 인치당 몇개의 점으로 이루어졌는지

- ppi(pixel per inch) : 인치당 몇개의 픽셀로 이루어졌는지

  > **dpi가 높을수록 화질 up** 

  

### 파일종류

- jpg  : 이미지파일. 배경o
- png : 이미지파일. 배경x. 인쇄x
- psd : 포토샵 전용파일. 2GB 제한
- psb : 대용량 포토샵 저장파일
- ai    : 일러스트레이터 전용파일
- eps : 일러스트레이터 인쇄용 
- pdf : 범용 저장형식 [모든 파일 열 수 있음]
- svg : ai파일 코드화
