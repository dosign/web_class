div는 의미없는, 단순히 영역을 나타나게 하는 박스

span 역시 의미없는 단순한 내용의 그룹

시멘틱 구조

영역을 나타내는 의미를 가진 구조

시멘틱 구조에 대해서 알아보자

header는 웹페이지 전체 구조를 파악할 수 있는 영역으로 로고/h1/nav를 포함

navigation

nav는 웹페이지 내부에서 클릭하여 해당페이지로 이동하는 모든 링크들의 집합

main는 header,footer 제외하고 감싸서 더 큰 영역에 담는 곳

하나의 대표영역

section이란 하나의 대표영역으로 다른 내용들과 연계되어 있는 내용들로 구성

연계되어 있지 않아도 되는 독립영역

article은 section과 유사한 기능, 대신 연계되어 있지 않아도 되는 독립영역으로 사용가능

another영역

aside는 없어도 되지만 있다면 도움이 되는 영역

사이트정보

footer는 회사or사이트 정보를 담는 영역으로 주로 하단에 존재함. 내용이 보이지 않아도 반드시 존재해야함
